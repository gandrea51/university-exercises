# University Exercises

Collection of programming exercises and course projects completed during my Bachelor's and Master's studies in Computer Science.

This repository serves as a personal archive of coursework developed throughout different university courses.

## Repository Structure

| Folder                     | Description                                                                  |
| -------------------------- | ---------------------------------------------------------------------------- |
| `architetture-hpc`         | Exercises on parallel programming and High Performance Computing.            |
| `computer-vision`          | OpenCV and PyTorch exercises for computer vision.                            |
| `fondamenti-data-science`  | MATLAB exercises related to data analysis and machine learning fundamentals. |
| `linguaggi-e-traduttori`   | Haskell exercises and programming language concepts.                         |
| `ottimizzazione-vincolata` | Constraint optimization exercises developed with Eclipse and ASP.            |
| `project-computer-vision`  | Course project focused on car segmentation using computer vision techniques. |
| `project-deep-learning`    | Course project on traffic sign classification (GTSRB) using deep learning.   |

## Notes

- All source code has been developed for educational purposes.
- The repository contains only my own implementations and project work.
- Lecture slides, exam papers, assignment statements, and other copyrighted course material are intentionally not included.
